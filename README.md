# Task 1: Hospital Landing Page

## Overview
This project is a responsive hospital landing page for **CityCare**. It presents the hospital brand, a clear hero section, featured services, department highlights, and a simple navigation experience built with plain HTML and CSS.

The focus of this task was clean visual hierarchy, accessibility-friendly structure, and a polished first impression suitable for a healthcare website.

## Tech Stack
- HTML5
- CSS3
- Bootstrap Icons
- Google Fonts

## Project Structure
- `index.html` - main landing page markup
- `style.css` - all page styling and responsive layout rules
- `logo.png`, `hospital.png`, `hospital-ui.png`, `doctors.webp` - visual assets used on the page

## Features
- Responsive header with navigation links
- Hero section with primary call-to-action buttons
- Service highlight cards for key hospital offerings
- About section with supporting image and copy
- Department showcase section

## How To Run
Run a local static server from the project folder:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Notes
This task is intentionally frontend-only and does not require a backend or build step.
